# ST. CLARET PORTAL

## Overview

The **St. Claret Portal** is a web-based recruitment management system developed for St. Claret College. The portal helps the college administration post teaching job openings and allows teachers/applicants to apply online through a simple and user-friendly platform.

The project aims to digitalize the recruitment process by reducing manual paperwork and making job application management more efficient.

---

## Features

* Admin Login System
* Post and Manage Job Openings
* Teacher/Applicant Registration
* Online Job Application Submission
* View Available Vacancies
* Database Management for Applications
* Responsive and User-Friendly Interface

## Installation Steps

1. Install a local server such as XAMPP or WAMP.
2. Copy the project folder into the `htdocs` directory.
3. Start Apache and MySQL from XAMPP/WAMP.
4. Open phpMyAdmin.
5. Create a new database.
6. Import the SQL file from the database folder.
7. Open the browser and run:

`http://localhost/St-Claret-Portal`
